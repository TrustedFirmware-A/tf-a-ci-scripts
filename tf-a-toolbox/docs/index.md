# TF-A Toolbox

The TF-A Toolbox is a CLI and Python API for small, scriptable tasks around Trusted Firmware-A testing and repository maintenance.

- See the [CLI Reference](cli-reference.md) for documentation on using the `tf-a-toolbox` command-line application.
- See the [Python API Reference](reference/tf_a_toolbox/index.md) for documentation on using the `tf_a_toolbox` Python API to build automation workflows.
